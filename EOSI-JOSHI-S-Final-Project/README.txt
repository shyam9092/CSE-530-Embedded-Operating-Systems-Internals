Open the linux terminal in the directory where usbtest.c is present and follow the below given test

$ sudo sudo apt-get install libusb-1.0

$ make 

Connect FTDI cable to your host machine

$ sudo ./usbtest

You will get the following output on terminal

Productid 	: 6001
Vendorid 	: 403
Manufacturer 	: FTDI
Product 	: FT232R USB UART
SerialNumber 	: A5065B7P
